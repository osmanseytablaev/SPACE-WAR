# SPACE-WAR

## Requirements
OS Linux Ubuntu >= 20.04

Python >= 3.8.5

##Install
```bash
     sudo apt-get install python-pygame
```
## Run game
```bash
    python3 space_war.py
```


## Rules game

Управление: стрелками;

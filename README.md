# SPACE-WAR

## Install
NOTES:
OS UBUNTU LINUX

```bash
    Install Python
    sudo apt install python3-pip
    
    Install Pygame
    sudo apt-get install python-pygame
    
```

## Run game
```bash
    python3 space_war.py
```


## Rules game
```bash
    Управление: стрелками;
    Стрельба: пробел.
    
    У тебя есть 3 жизни.
    Если ты набрал 30 очков ты победил.
```
